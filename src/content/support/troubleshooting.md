## I cannot sign in
Ask the user to confirm they are using the same email address they used to create the account. They can also try a fresh browser session, clear cookies for WHOMA, or request a new sign-in or reset email if available.

If the issue continues, tell them a real person can help and that they can click "Talk to a person" in the chat.

## I did not receive an email
Ask the user to check spam or junk folders, confirm the address was typed correctly, and wait a few minutes.

If the email still does not arrive, they can click "Talk to a person" so support can inspect the issue.

## My profile will not publish
Ask the user to check for missing required fields, incomplete onboarding steps, unverified work email, or pending review status.

Agents should make sure service areas, specialties, bio, and contact details are accurate before publishing.

## My past deal verification email did not arrive
Ask the agent to confirm the seller email address is correct and wait a few minutes for delivery.

If it still does not arrive, support can help check the verification request.

## The website looks stuck or a button does not respond
Ask the user to refresh the page, sign out and back in, or try a different browser. If the problem persists, ask them to share the page they were on and what they clicked.

## I found wrong information
Ask the user not to rely on inaccurate information. They can edit their own profile information if they are the agent, or contact support if the issue concerns public evidence or verification.
